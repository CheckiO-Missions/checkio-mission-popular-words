You need to break the input text into single words and calculate the number of each word (that we are looking for according to the given data) in the resulted list of words. Then you need to create a dictionary and add the information to it.
OK, let's start from the beginning.
First of all, you need to break the input text into single words. For this you can use the str.split() method.
How will it look like in code?
At this stage, the function will be as follows: 
def popular_words(text, words):
    splitted_words = text.split()
In practice, this method will work in the following way: 'I am alive'.split() --> ['I', 'am', 'alive']
How to count in the resulting list the number of words that we are looking for and add this information to the dictionary?
To count the number of each word from the input 'words' list, you must use 'for' loop. You can read more about the calculation with the 'count' method in the following catalogue of methods of working with lists.
How to implement these methods in code?
The updated function will look like this: 
def popular_words(text, words):
    splitted_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
This will create the answer keys inside the dictionary with words from the 'words' list, and assign them values that will show how many times each word is found in the input text broken into single words (splitted_words).
What to do if "one" and "One" are considered to be different words?
You need to replace all of the uppercase letters in the input text with the lowercase letters. In this situation will help the str.lower() method.
How will the whole function look like?
After adding the corresponding string, code will look like this: 
def popular_words(text, words):
    text = text.lower()
    splitted_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
Note that you must first convert all of the uppercase letters to the lowercase letters in the text, and then break it into the single words.
