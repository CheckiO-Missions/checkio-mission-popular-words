In dieser Mission ist es Ihre Aufgabe, die Beliebtheit bestimmter Wörter im Text zu ermitteln.

Bei der Eingabe Ihrer Funktion erhalten Sie 2 Argumente: den Text und das Array von Wörtern, deren Beliebtheit Sie bestimmen müssen.

Achten Sie beim Lösen dieser Aufgabe auf die folgenden Punkte:

  • Die Wörter sollten in allen Registern gesucht werden. Das heißt, wenn Sie ein Wort "eins" finden müssen, dann sind Wörter wie "eins", "One", "oNe", "ONE" usw. ausreichend.
  • Die Suchwörter werden immer in Kleinbuchstaben angegeben.
  • Wenn das Wort nicht ein einziges Mal gefunden wurde, muss es im Wörterbuch mit dem Wert 0 (Null) zurückgegeben werden.

Eingabe: Der Text und das Array mit den Suchwörtern.

Ausgabe: Das Wörterbuch, in dem die Suchwörter die Schlüssel sind und die Werte die Anzahl der Vorkommen dieser Wörter in einem bestimmten Text.
