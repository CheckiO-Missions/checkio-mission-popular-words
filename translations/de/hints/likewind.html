Sie müssen den Eingabetext in einzelne Wörter zerlegen und die Anzahl der einzelnen Wörter (nach denen wir suchen) berechnen nach dem wir suchen) in der resultierenden Wortliste. Dann müssen Sie ein Wörterbuch erstellen und ihm die Informationen hinzufügen.
OK, lassen Sie uns von vorne beginnen.
Zunächst müssen Sie den eingegebenen Text in einzelne Wörter zerlegen. Hierfür können Sie die Option str.split() method.
Wie sieht das im Code aus?
In diesem Stadium wird die Funktion wie folgt aussehen: 
def popular_words(text: str, words: list) -> dict:
    splitted_words = text.split()
In der Praxis wird diese Methode folgendermaßen funktionieren: 'Ich bin am Leben'.split() --> ['Ich', 'bin', 'am Leben']
Wie kann man in der resultierenden Liste die Anzahl der gesuchten Wörter zählen und diese Information dem Wörterbuch hinzufügen?
Um die Anzahl der einzelnen Wörter aus der Eingabeliste 'Wörter' zu zählen, müssen Sie 'for' loop. Sie können mehr lesen über die Berechnung mit der Methode 'count' in dem folgenden Katalog von methods of working with lists.
Wie implementiert man diese Methoden in Code?
Die aktualisierte Funktion wird wie folgt aussehen: 
def popular_words(text: str, words: list) -> dict:
    splitted_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
Sie erstellt die Antwortschlüssel innerhalb des Wörterbuchs mit Wörtern aus der Liste 'words' und weist ihnen Werte zu, die zeigen wie oft jedes Wort im Eingabetext in einzelne Wörter zerlegt vorkommt (splitted_words).
Was ist zu tun, wenn "eins" und "One" als unterschiedliche Wörter betrachtet werden?
Sie müssen alle Großbuchstaben im Eingabetext durch Kleinbuchstaben ersetzen. In dieser Situation hilft Ihnen die str.lower() method.
Wie wird die gesamte Funktion aussehen?
Nachdem Sie die entsprechende Zeichenkette hinzugefügt haben, sieht der Code wie folgt aus: 
def popular_words(text: str, words: list) -> dict:
    text = text.lower()
    splitted_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
Beachten Sie, dass Sie zunächst alle Großbuchstaben in die Kleinbuchstaben des Textes umwandeln müssen, und ihn dann in einzelne Wörter aufteilen.
