Вам необходимо создать словарь, затем посчитать количество каждого слова в исходном тексте (из тех, что мы ищем согласно исходным данным) и добавить эту информацию в созданный словарь.
Я не знаю, как посчитать слова и добавить эту информацию в словарь.
Если нам нужно посчитать количество слов 'one', нужно набрать: 
def popular_words(text, words):
    answer = {}
    answer['one'] = text.count('one')
    return answer
Это создаст внутри словаря answer ключ 'one' и присвоит ему значение, которое будет показывать, сколько раз слово 'one' встречается в исходном тексте.
Но ведь нам нужно посчитать не только количество слова 'one'. Что делать с остальными словами?
Для перебора остальных слов из исходного списка, используйте 'for' loop.
Вы могли бы показать, как именно это осуществить?
Код, написанный ранее, меняем следующим образом: 
def popular_words(text, words):
    answer = {}
    for word in words:
        answer[word] = text.count(word)
    return answer
В этом случае в итоговый словарь 'answer' в качестве ключей добавятся все слова, количество которых мы считаем, согласно исходному списку 'words', а в качестве значений к этим ключам - их количество в 'text'.
Что делать, если ответ все равно получается неправильным?
Скорее всего вы не учли, что в исходном тексте есть слова, написанные буквами и верхнего, и нижнего регистра. Чтобы функция возвращала верный результат - в исходном тексте необходимо преобразовать все буквы верхнего регистра в буквы нижнего. В этой ситуации поможет метод str.lower().
Как именно пользоваться этим методом?
Нужно добавить в код строчку: 
def popular_words(text, words):
    text = text.lower()
    answer = {}
    for word in words:
        answer[word] = text.count(word)
    return answer
Почему в тексте "I love it!" функция выдает 'i': 2 вместо 'i': 1?
Это происходит потому, что буква 'i' входит в слово 'it' и функция считает, что слово 'it' тоже удовлетворяет критериям поиска. Чтобы избежать этого, необходимо разбить весь текст на отдельные слова с помощью метода split().
Как изменится код после этого?
На данном этапе функция будет выглядеть так: 
def popular_words(text, words):
    text = text.lower()
    splited_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splited_words.count(word)
    return answer
Что делать, если некоторые слова в списке выглядят как 'one!' или 'one.' и не считаются за 'one'?
Похоже, вы забыли убрать из текста лишние символы - знаки препинания ('.', ',', '!', '?'). В этом вам поможет .
Как мне интегрировать этот метод в существующий код?
Вам нужно заменить знаки препинания на пробелы - ' ' (но не на пустые строки - '', чтобы не допустить слияния слов в одно, например - 'hello.I am'.replace('.', '') --> 'helloI am'). Поэтому ваш код будет выглядеть так: 
def popular_words(text, words):
    text = text.replace(',', ' ')
    text = text.replace('.', ' ')
    text = text.replace('!', ' ')
    text = text.replace('?', ' ')
    text = text.lower()
    splited_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splited_words.count(word)
    return answer
Это же можно записать чуть меньшим количеством строк: 
def popular_words(text, words):
    text = text.replace(',', ' ').replace('.', ' ').replace('!', ' ').replace('?', ' ')
    text = text.lower()
    splited_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splited_words.count(word)
    return answer
