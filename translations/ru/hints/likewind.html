У меня совершенно нет идей. Помогите!
Вам необходимо создать словарь, затем посчитать количество каждого слова в исходном тексте (из тех, что мы ищем согласно исходным данным) и добавить эту информацию в созданный словарь.
Давайте по-порядку.
Сперва необходимо разделить исходный текст на отдельные слова. Для этого можно использовать метод split(). Если этого не сделать, могут возникнуть ошибки подсчета, когда, например 'I am alive'.count('a') возвращает значение 2, так как буква 'a' дважды входит в состав других слов, хотя отдельного слова 'a' в этой строке нет.
Как это будет выглядеть в коде?
На данном этапе функция будет выглядеть так: 
def popular_words(text, words):
    splitted_words = text.split()
Как посчитать в получившемся списке количество слов, которые мы ищем и добавить эту информацию в словарь?
Для подсчета количества каждого слова из исходного списка 'words' необходимо использовать 'for' loop. О подсчете с помощью метода 'count' вы сможете прочесть подробнее в перечне методов работы со списками
Как эти методы реализовать в коде?
Обновленная функция будет выглядеть так: 
def popular_words(text, words):
    splitted_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
Это создаст внутри словаря answer ключи со словами из списка 'words' и присвоит им значения, которые будут показывать, сколько раз каждое из слов встречается в исходном тексте, поделенном на слова (splitted_words).
Что делать, если рядом с некоторыми словами находятся знаки препинания, например 'one!' или 'one.' и такие слова не считаются эквивалентными 'one'?
Необходимо убрать из текста лишние символы - знаки препинания ('.', ',', '!', '?'). В этом вам поможет метод str.replace().
Как мне интегрировать этот метод в существующий код?
Вам нужно заменить знаки препинания на пробелы - ' ' (но не на пустые строки - '', чтобы не допустить слияния слов в одно, например - 'hello.I am'.replace('.', '') --> 'helloI am'). Поэтому ваш код будет выглядеть так: 
def popular_words(text, words):
    splitted_words = text.split()
    text = text.replace(',', ' ')
    text = text.replace('.', ' ')
    text = text.replace('!', ' ')
    text = text.replace('?', ' ')
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
Это же можно записать чуть меньшим количеством строк: 
def popular_words(text, words):
    splitted_words = text.split()
    text = text.replace(',', ' ').replace('.', ' ').replace('!', ' ').replace('?', ' ')
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
Что делать, если 'one' и 'One' считаются разными словами?
Необходимо заменить в исходном тексте все буквы верхнего регистра буквами нижнего. В этой ситуации поможет метод str.lower().
Как именно пользоваться этим методом?
Нужно добавить в код соответствующую строчку, после чего он станет таким: 
def popular_words(text, words):
    splitted_words = text.split()
    text = text.replace(',', ' ').replace('.', ' ').replace('!', ' ').replace('?', ' ')
    text = text.lower()
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
