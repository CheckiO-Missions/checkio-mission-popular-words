Вам необходимо разбить исходный текст на отдельные слова и посчитать количество каждого слова (из тех, что мы ищем согласно исходным данным) в получившемся списке слов, а затем создать словарь и добавить эту информацию в него.
Давайте по-порядку.
Сперва необходимо разделить исходный текст на отдельные слова. Для этого можно использовать метод split().
Как это будет выглядеть в коде?
На данном этапе функция будет выглядеть так: 
def popular_words(text, words):
    splitted_words = text.split()
На практике этот метод будет работать следующим образом: 'I am alive'.split() --> ['I', 'am', 'alive']
Как посчитать в получившемся списке количество слов, которые мы ищем и добавить эту информацию в словарь?
Для подсчета количества каждого слова из исходного списка 'words' необходимо использовать 'for' loop. О подсчете с помощью метода 'count' вы сможете прочесть подробнее в перечне методов работы со списками
Как эти методы реализовать в коде?
Обновленная функция будет выглядеть так: 
def popular_words(text, words):
    splitted_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
Это создаст внутри словаря answer ключи со словами из списка 'words' и присвоит им значения, которые будут показывать, сколько раз каждое из слов встречается в исходном тексте, поделенном на слова (splitted_words).
Что делать, если 'one' и 'One' считаются разными словами?
Необходимо заменить в исходном тексте все буквы верхнего регистра буквами нижнего. В этой ситуации поможет метод str.lower().
Как будет выглядеть функция полностью?
После добавления соответствующей строчки, код станет таким: 
def popular_words(text, words):
    text = text.lower()
    splitted_words = text.split()
    answer = {}
    for word in words:
        answer[word] = splitted_words.count(word)
    return answer
Обратите внимание, что необходимо сперва преобразовать в тексте все буквы верхнего регистра в нижний, а затем делить на отдельные слова.
